### Crypto-tracker website via Python(Fast.API) + JS(React.js)
 ### To run it, you need to run the back-end and front-end code in separate terminals 
 
## front
- `cd front-end`
- `npm run dev`



## back

- `cd back-end`
- `python -m venv venv`
- `. ./venv/bin/activate`
- `pip install -r requirements.txt`
- `cd src`
- `python main.py`
